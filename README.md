# QuranCounter
This project will count occurrence of all characters in each sura.

In order to use, find the sura you want here: 
http://quran.roshd.ir/portal/Home/Default.aspx?CategoryID=e906b14c-b221-4fe3-96a5-1ff248053d24

Then select the whole text and paste it in “amar.txt” file in \bin\Debug directory and save it.

Now run the program and create the pdf output by click on the button.
